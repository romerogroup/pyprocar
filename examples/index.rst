PyProcar Examples
==================

Welcome to the PyProcar examples! This collection of Jupyter notebooks demonstrates various use cases and practical applications of PyProcar, a Python library for electronic structure pre/post-processing.

These examples are automatically generated from the `examples directory`_ of the package and showcase how to effectively use PyProcar's features for plotting band structures, density of states, Fermi surfaces, and more. Feel free to download and run these notebooks to explore the functionality firsthand.
