Plotting spin-polarized density of states
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Plotting spin-polarized density of states example.

First download the example files with the code below. Then replace data_dir below.

    data_dir = pyprocar.download_example(save_dir='',

Plain mode
+++++++++++++++

When the calculation is a spin-polarized calculation. There are few more features features bandsplot can do. 
The default settings bandsplot will plot the spin-up and spin-down bands on the same plot.

.. GENERATED FROM PYTHON SOURCE LINES 38-43

.. code-block:: default

    pyprocar.dosplot(
                    code='qe', 
                    mode='plain',
                    dirname=data_dir)

Parametric mode
+++++++++++++++

For details on the meaning of the indices of the atomic projection please refer to the user guide :ref:'atomic_projections'




.. GENERATED FROM PYTHON SOURCE LINES 62-76

.. code-block:: default

    atoms=[0]
    orbitals=[4,5,6,7,8]
    spins=[0,1]

    pyprocar.dosplot(
                    code='qe', 
                    mode='parametric',
                    atoms=atoms,
                    orbitals=orbitals,
                    spins=spins,
                    vmin=0,
                    vmax=1,
                    dirname=data_dir)
