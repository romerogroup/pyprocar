Plain mode
+++++++++++++++++++++++++++++++++++++++

When the calculation is a spin-polarized calculation. There are few more features features bandsplot can do. 
The default settings bandsplot will plot the spin-up and spin-down bands on the same plot.

.. GENERATED FROM PYTHON SOURCE LINES 37-42

.. code-block:: default

    pyprocar.dosplot(
                    code='vasp', 
                    mode='plain',
                    dirname=data_dir)

Parametric mode
+++++++++++++++++++++++++++++++++++++++

For details on the meaning of the indices of the atomic projection please refer to the user guide :ref:'atomic_projections'




.. GENERATED FROM PYTHON SOURCE LINES 61-75

.. code-block:: default

    atoms=[0]
    orbitals=[4,5,6,7,8]
    spins=[0,1]

    pyprocar.dosplot(
                    code='vasp', 
                    mode='parametric',
                    atoms=atoms,
                    orbitals=orbitals,
                    spins=spins,
                    vmin=0,
                    vmax=1,
                    dirname=data_dir)
